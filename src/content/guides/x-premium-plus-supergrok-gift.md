## 先看结论

X Premium 礼物订阅是官方的“替别人一次性购买一年会员”功能，**不是折扣活动**。

你需要先是 X Premium 用户，再通过网页端进入一个尚未订阅 Premium 的账号主页，选择赠送 Premium 或 Premium+。礼物只能买年付，付款后立即开始计时，不自动续费，也不能退款或转给其他账号。

原稿标题中的“附 SuperGrok 权益”不能作为固定结论。X 当前对个人 Premium+ 明确写的是“更高的 Grok 使用上限”；独立 SuperGrok 是否到账，要以接收账号在 grok.com 的计划页面实际显示为准。

## Premium 与 Premium+ 怎么选

| 项目 | Premium | Premium+ |
| --- | --- | --- |
| 美国网页年费参考 | US$84 | US$395 |
| 蓝色标记 | 需通过账号审核 | 需通过账号审核 |
| 广告 | For You / Following 时间线约减少 50% | X 大部分区域免广告，仍可能出现少量赞助内容 |
| Grok | 提高使用上限 | 更高使用上限 |
| 回复优先级 | 较高 | 最高 |
| X Pro | 不包含 | 包含 |
| 适合人群 | 主要需要认证、创作者功能和基础 Grok 增量 | 高频使用 X、需要更少广告和更高 Grok 上限 |

实际权益会变化。付款前必须打开当前结算页和官方权益页，不要只根据旧截图决定购买。

## 完整领取流程

1. 赠送者在浏览器登录 x.com，并确认自己的 Premium 有效。
2. 打开接收者个人主页，确认 @用户名准确，且对方当前没有 Premium。
3. 点击个人主页上的礼物图标。
4. 选择 Premium 或 Premium+；当前只提供年度礼物。
5. 在 Gift & Pay 页面核对全年价格、币种、税费和接收账号。
6. 完成付款后，让接收者查看 X 通知并确认条款。
7. 接收者检查 Premium 档位、到期日期、蓝色标记审核状态和 Grok 实际权益。
8. 双方保存订单、通知和到账截图。

## Premium+ 和 SuperGrok 的关系

这两个名称不要混用：

- **X Premium+** 是 X 的最高个人订阅档位，核心权益包括更高 Grok 上限、X 大部分区域免广告、X Pro、最高回复优先级和 Radar Search。
- **SuperGrok** 是 grok.com / Grok App 使用的独立高级订阅名称，官方描述为更高上限、优先访问和 Multi-agent 等权益。
- X 官方对 Premium Business、Premium Organizations 及其附属账号明确写过包含 SuperGrok，但个人 Premium+ 的通用权益页只明确承诺更高 Grok 上限。

因此，个人 Premium+ 礼物到账后，应使用接收账号打开 X 内 Grok 和 grok.com 计划页面实测。页面没有明确显示 SuperGrok 时，就不要对外宣传“Premium+ 免费送 SuperGrok”。

## 常见失败排查

| 问题 | 优先检查 |
| --- | --- |
| 接收者主页没有礼物图标 | 是否在 x.com 网页端、赠送者 Premium 是否有效、接收者是否已有订阅 |
| 付款后接收者没看到通知 | 是否送错 @用户名、通知是否被折叠、订单是否真正成功 |
| 没有蓝色标记 | 等待账号资格和安全审核；近 3 天是否改过头像、名称或用户名 |
| 礼物被取消 | 接收者是否未确认条款、不符合资格、主动取消或账号受限 |
| Grok 上限没有变化 | 是否登录了相同 X 账号、档位是否确为 Premium+、权益是否仍在灰度开放 |
| 想退款或改送其他账号 | 官方礼物订单不可退款、不可替换，也不能转移接收者 |
