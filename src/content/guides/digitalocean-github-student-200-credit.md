## 先看结论

GitHub Student Developer Pack 当前向已验证学生提供 **US$200 DigitalOcean 平台赠金**，GitHub Education 页面标注优惠持续至 **2026 年 7 月 31 日**。

这不是“无卡、永久免费”的学生服务器。领取和创建资源前需要绑定有效付款方式，赠金过期、用完或遇到不适用赠金的费用后，DigitalOcean 会按正常账单向默认付款方式收费。

## 当前不能用赠金抵扣的重点项目

GitHub Education 当前列出的排除项目包括：

- GPU Droplets
- NVIDIA H100 GPU Kubernetes
- Bare Metal GPUs
- Dedicated Inference
- 部分 Serverless Inference
- 托管在 DigitalOcean 基础设施之外的第三方 AI 模型费用
- Anthropic、OpenAI 等外部商业或前沿模型费用
- Paperspace Machines
- Cloudways 服务
- 其他由 DigitalOcean 转付给第三方的费用

普通 Droplet、App Platform、Managed Databases、Spaces 等是否适用，应以兑换后的 Billing 和资源结算页面为准。

## 最稳妥的领取顺序

1. 确认 GitHub Student Developer Pack 已通过验证。
2. 从 GitHub Education 官方页面进入 DigitalOcean 专属领取入口。
3. 登录或创建 DigitalOcean 账号。
4. 添加官方接受的付款方式。
5. 在 Billing 页面确认 US$200 Credit、到期日和适用范围。
6. 开启 Billing Alert。
7. 只创建一个最低配置测试资源。
8. 每隔几天检查余额和账单。
9. 到期前导出数据并销毁所有不再使用的资源。

## 防止意外扣费

- 不要把 Billing Alert 当成自动停机工具。
- 不要只关机，Droplet 必须 Destroy 才停止计算计费。
- 删除主机后继续检查磁盘、快照、数据库、负载均衡和对象存储。
- 不要默认 AI 模型费用可以用学生赠金抵扣。
- 到期前主动清理，赠金失效不会自动关闭账号和资源。
