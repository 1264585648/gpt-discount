## 先看结论

Azure for Students 是一项真正适合学生练习云计算的官方赠金：**注册时不需要信用卡，首个 12 个月可获得 US$100 Azure 额度**，并可使用多项首年免费或长期免费服务。

它的核心安全边界是：

- 没有主动升级为 Pay-As-You-Go 时，额度用完会停用订阅，不会自动扣款。
- 主动升级后会进入正常按量付费，未删除的资源可能持续产生费用。
- “免费服务”都有规格、地区和月度用量限制，虚拟机免费不代表磁盘、公网 IP 和流量也全部免费。
- Azure OpenAI 和 Foundry 可以出现在学生产品页，但模型、地区、配额和调用价格仍需按账号实际页面确认。

## 最稳妥的领取顺序

1. 确认自己年满 18 岁，并且是合格高校的全日制学生。
2. 从 Azure for Students 页面进入，不要误开普通 Azure 免费试用。
3. 用学校邮箱完成学生验证。
4. 进入 Portal 确认订阅类型和 US$100 到期日。
5. 创建任何资源之前先设置预算提醒。
6. 先从 Static Web Apps、Functions 等低风险资源测试。
7. 用完后删除整个资源组，避免遗留磁盘、公网 IP 和快照。

## 赠金能用在哪里

US$100 可以用于大多数 Azure 自营云服务，例如虚拟机、数据库、存储、应用托管和部分 AI 服务，但不是现金，也不能转给其他账号。

以下项目官方明确可能无法使用赠金支付：

- Azure Marketplace 第三方产品
- Azure 支持计划
- Azure DevOps 付费项目
- Visual Studio 订阅
- ExpressRoute
- 第三方品牌产品或单独授权产品
- Microsoft Entra ID Premium 等部分独立许可

## 新手最容易产生费用的地方

| 场景 | 容易忽略的费用 |
| --- | --- |
| 创建虚拟机 | 托管磁盘、公网 IP、快照、出站流量、备份 |
| 只点击 Stop | 部分资源仍保留并继续计费，应检查是否已 Deallocate |
| 删除虚拟机 | 磁盘、网卡和公网 IP 可能没有一起删除 |
| 创建数据库 | 免费层有计算、存储和地区限制 |
| 部署 AI 模型 | 模型调用按 Token、字符、图片或请求量计费 |
| 设置预算 | 预算只提醒，不会自动停止服务 |

测试项目结束后，最简单的清理方法是进入 **Resource groups**，确认没有重要数据，再删除整个测试资源组。
