## 官方入口

- 学生认证：https://github.com/settings/education/benefits
- Student Developer Pack：https://education.github.com/pack
- GitHub Education：https://github.com/education

## 提醒

学生包中的合作权益会调整。领取前查看具体产品的地区、期限、续费和付款规则。Copilot 是否重新开放，以 Student Developer Pack 页面为准。
